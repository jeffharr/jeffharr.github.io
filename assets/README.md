# Soundtrack

Drop the vaporwave loop here as `soundtrack.mp3` (or `soundtrack.ogg`).
The player on the site references `assets/soundtrack.mp3` first, then `.ogg`.

- Loops automatically, starts on user click (browsers block autoplay).
- Keep it a clean loop (no hard start/stop) so it cycles seamlessly.
